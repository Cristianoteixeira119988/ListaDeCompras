A aplicação terá 3 botoes iniciais, um de adicionar produto a uma lista, outro de ver todas as litas completas de produtos e outro para criar uma nova lista.
Quando se clica em adicionar um produto deverá aparecer para escolher a lista a qual deseja adicionar esse produto (porque cada lista tem uma categoria: mercearia,bricolage e vestuario),
depois de selecionada a lista devera aparecer uma nova activity para registar o novo produto, com nome, quantidade, se quer uma marca especifica ou o produto mais barato
e a data em que o produto ficou em falta para assim o utilizador saber se é melhor comprar o produto em mais quantidade ou nao.

Listas: id_lista, categoria, nome_lista, data_criacao

Produtos: id_produto, categoria, quantidade, data_que_acabou

Categorias: id_categoria, categoria

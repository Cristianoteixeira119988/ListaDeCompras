A aplicação terá 4 botoes iniciais, um para visualizar todos os produtos, outro de ver todas as litas, outro para criar uma nova lista e 
por ultimo um para gerir as categorias.
Quando se clica para ver todos os produtos deverá aparecer para escolher a lista para inserir/ editar/ eliminar um produto,
depois de selecionada a lista devera aparecer uma nova activity para registar o novo produto/eliminar o produto, com nome,  
quantidade e a data em que o produto ficou em falta para assim o utilizador saber se é melhor comprar o produto em mais quantidade ou nao.

Listas: id_lista, nome_lista, data_criacao

Produtos: id_produto, nome_produto, categoria, quantidade, data_que_acabou

Categorias: id_categoria, categoria

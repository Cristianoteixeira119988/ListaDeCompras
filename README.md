A aplicação terá 2 botoes iniciais, um de adicionar produto á lista e outro de ver a lita completa de produtos a comprar.
Quando se adicionar um produto deverá aparecer para colocar a quantidade que depois deseja comprar,
deverá aparecer tambem se deseja um produto barato ou de uma marca especifica e a data em que este
produto ficou em falta para assim o usuario saber se é melhor comprar o produto mais quantidade ou nao.
